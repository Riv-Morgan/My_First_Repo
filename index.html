ABOUT THE PROJECT...

Snowut.com is built to deliver information to snowmobilers about public lands issues, avalanche dangers, and grooming reports. The current website is not always accurate nor easy to navigate. We determined that by overhauling the site we would be able to get more accurate information to riders, faster, and more effectively. This will allow riders to be safe, make friends, and give back to their sport.
